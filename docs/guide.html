IRQ Optimizer 상세 사용 가이드

Windows 스케줄러의 추측 기반 분산 대신, 핵심 장치 IRQ를 직접 지정해 입력 지연과 프레임타임 흔들림을 완화하는 실전 절차입니다.

1) 실행 준비

  • Windows 관리자 권한으로 실행합니다.
  • python irq_optimizer_ultimate_safe.py로 시작합니다.
  • 권한이 낮으면 UAC 재실행이 자동 시도됩니다.

2) 장치 스캔

  • Refresh Devices 클릭
  • 역할 태그(GPU/GPU-ROOT/AUDIO/STORAGE/NIC) 확인
  • Select Target Devices로 권장 대상 자동 선택

3) 코어 지정

  • Use Recommended로 권장 코어 자동 선택
  • 필요하면 체크박스로 수동 조정
  • 장치별 현재 마스크가 있으면 선택 시 표시됩니다.

4) 적용 및 검증

  • Apply로 레지스트리에 마스크 기록
  • 적용 전 상태는 자동 백업됩니다.
  • 쓰기 후 즉시 검증되어 실패 시 에러를 표시합니다.

5) 복구

  • Undo Last: 마지막 변경 되돌리기
  • Factory Reset: 커스텀 IRQ 값 제거
  • 백업 파일: C:\ProgramData\IRQOptimizer\irq_backup.json

운영 권장

  • 권장GPU/오디오부터 순차 적용
  • 권장적용 후 프레임타임/입력지연 비교
  • 주의시스템별 편차가 존재

참고: GitHub README에는 커스텀 CSS 제약이 있으므로, 스타일 문서는 별도 HTML/CSS로 제공하는 방식이 가장 안정적입니다.
