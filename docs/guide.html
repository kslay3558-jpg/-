IRQ Optimizer 상세 사용 가이드

Windows 스케줄러의 추측 기반 분산 대신, 핵심 장치 IRQ를 직접 지정해 입력 지연과 프레임타임 흔들림을 완화하는 실전 절차입니다.

1) 실행 준비

  • Windows 관리자 권한으로 실행합니다.
  • python irq_optimizer_ultimate_safe.py로 시작합니다.
  • 권한이 낮으면 UAC 재실행이 자동 시도됩니다.

2) 장치 스캔

  • Refresh Devices 클릭
  • 역할 태그(GPU/GPU-ROOT/AUDIO/STORAGE/NIC) 확인
  • Select Target Devices로 권장 대상 자동 선택

3) 코어 지정

  • Use Recommended로 권장 코어 자동 선택
  • 필요하면 체크박스로 수동 조정
  • 장치별 현재 마스크가 있으면 선택 시 표시됩니다.

4) 적용 및 재부팅

  • Apply로 레지스트리에 마스크 기록
  • 적용 전 상태는 자동 백업됩니다.
  • 쓰기 후 즉시 검증되어 실패 시 에러를 표시합니다.
  • 중요Apply 후 반드시 시스템을 재부팅해야 변경이 실제로 적용됩니다.

5) 복구

  • Undo Focused: 현재 포커스 장치의 마지막 변경 되돌리기
  • Reset Focused: 현재 포커스 장치의 커스텀 IRQ 값 제거
  • 백업 파일: C:\ProgramData\IRQOptimizer\irq_backup.json
  • 중요복구 후에도 재부팅해야 실제로 적용됩니다.

토폴로지 기반 추천 엔진

  • 앱 실행 시 CPU 아키텍처를 자동 분류합니다.
  • 분기(Branch) 종류:
    • intel_hybrid – P-코어 클러스터에 GPU/Root Port 배치, E-코어에 사이드 장치
    • intel_legacy – 균일 물리 코어에 고르게 분산
    • amd_dual_x3d – 1차 CCD에 GPU, 2차 CCD에 사이드 장치
    • amd_single_x3d / amd_generic – X3D 캐시 인접성을 유지하며 분산
  • 주의추천값은 참고용입니다. BIOS 토폴로지 확인 후 수동 조정을 권장합니다.

운영 권장

  • 권장GPU/오디오부터 순차 적용
  • 권장적용 후 프레임타임/입력지연 비교
  • 주의시스템별 편차가 존재
