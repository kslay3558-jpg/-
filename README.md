# IRQ Optimizer (Ultimate Safe Edition)

Windows 환경에서 GPU, 오디오, 스토리지, NIC 등 핵심 장치의 **IRQ Affinity(인터럽트 친화도)**를 직접 지정해  
게임 중 입력 지연(Input Lag)과 프레임타임 흔들림(스터터링, 1% Low 저하)을 완화하는 도구입니다.

Windows 기본 스케줄러는 인터럽트를 모든 코어에 균일하게 분배하려 합니다.  
하지만 AMD 칩렛/Infinity Fabric 지연, Intel P/E 코어 혼재 환경에서는 이 방식이 오히려 레이턴시를 높일 수 있습니다.  
이 도구를 사용하면 특정 장치의 IRQ를 원하는 코어에 고정해 예측 가능한 성능을 얻을 수 있습니다.

> **주의**  
> - 시스템·드라이버·게임마다 효과의 편차가 크며, 모든 환경에서 동일한 개선을 보장하지 않습니다.  
> - 잘못된 코어 고정은 오히려 성능을 저하시킬 수 있습니다.  
> - **설정을 적용한 뒤에는 반드시 시스템을 재부팅해야 변경 사항이 실제로 적용됩니다.**  
> - 레지스트리를 수정하므로 문제 발생 시 `Undo Last` 또는 `Factory Reset`으로 즉시 되돌릴 수 있습니다.

---

## 요구 사항

- Windows 10 또는 Windows 11
- 관리자 권한 실행
- Python 3.9 이상

> `winreg` 모듈을 사용하므로 **Windows 전용**입니다. macOS·Linux에서는 동작하지 않습니다.

---

## 빠른 시작

1. **관리자 권한**으로 PowerShell 또는 CMD를 실행합니다.
2. 저장소를 받은 폴더로 이동합니다.
3. 아래 명령을 실행합니다.

```bash
python irq_optimizer_ultimate_safe.py
```

권한이 부족한 경우 UAC 프롬프트가 자동으로 나타납니다.

## 사용 방법

### 1단계 — 장치 스캔

1. 앱을 실행하면 장치 목록이 자동으로 불러와집니다.
2. 목록이 비어 있거나 새로고침이 필요하면 **Refresh Devices** 버튼을 클릭합니다.
3. 각 장치 앞의 역할 태그(`[GPU]`, `[GPU-ROOT]`, `[AUDIO]`, `[STORAGE]`, `[NIC]`, `[USB]`)를 확인합니다.

### 2단계 — 대상 장치 선택

- **자동 선택(권장):** `Target Preference Profile` 드롭다운에서 용도에 맞는 프로필을 고른 뒤 **Select Target Devices**를 클릭합니다.
  - `Balanced` : 게임·작업 혼용 환경
  - `Low Latency` : 경쟁 게임 등 낮은 지연 우선
  - `Streaming` : 방송/녹화 병행 환경
- **수동 선택:** 목록에서 장치를 직접 클릭해 선택합니다.

### 3단계 — CPU 코어 지정

- 장치를 선택하면 현재 설정된 마스크(있을 경우)가 체크박스에 표시됩니다.
- **Use Recommended** 버튼을 누르면 CPU 아키텍처 기반 권장 코어가 자동으로 선택됩니다.
- 필요하면 체크박스를 직접 조정합니다.

### 4단계 — 적용 및 재부팅

1. **Apply** 버튼을 클릭합니다.
2. 앱이 현재 상태를 자동 백업한 뒤 레지스트리에 `DevicePolicy=4`와 `AssignmentSetOverride` 값을 기록합니다.
3. 기록 즉시 검증이 이루어지며, 성공 또는 실패 메시지가 표시됩니다.
4. ⚠️ **성공 메시지가 표시된 뒤에도 실제 적용은 시스템 재부팅 후에 이루어집니다. 반드시 재부팅하세요.**

### 5단계 — 효과 확인

- 재부팅 후 게임을 10~20분 플레이하며 입력 지연과 프레임타임을 이전과 비교합니다.
- 효과가 없거나 오히려 나빠졌다면 아래 복구 방법을 사용하세요.

### 복구 방법

| 버튼 | 동작 |
|---|---|
| **Undo Last** | 선택한 장치의 마지막 변경을 이전 상태로 되돌립니다 |
| **Factory Reset** | 선택한 장치의 모든 커스텀 IRQ 값을 제거합니다 |

> 복구 후에도 **재부팅**해야 변경이 실제로 적용됩니다.

백업 파일 위치:

```text
C:\ProgramData\IRQOptimizer\irq_backup.json
```

## 토폴로지 기반 자동 추천 엔진

앱 시작 시 CPU 아키텍처를 자동으로 감지하고, 장치 역할에 맞는 코어 그룹을 추천합니다.

| 분기 | 해당 CPU | 배치 전략 |
|---|---|---|
| `intel_hybrid` | 12세대 이상 Core, Core Ultra | GPU·Root Port → P-코어 클러스터, 사이드 장치 → E-코어 |
| `intel_legacy` | 11세대 이하 Core | GPU → 하위 물리 코어, 사이드 장치 → 균일 분산 |
| `amd_dual_x3d` | Ryzen 7950X3D / 9950X3D 등 | GPU → 1차 CCD, 사이드 장치 → 2차 CCD |
| `amd_single_x3d` | Ryzen 5800X3D / 7800X3D 등 | GPU → X3D 캐시 인접 코어, 사이드 장치 → 나머지 코어 |
| `amd_generic` | 그 외 Ryzen / Threadripper | GPU → 하위 물리 코어, 사이드 장치 → 분리 배치 |

> 추천값은 참고용입니다. BIOS/드라이버 레벨 토폴로지를 완전히 반영하지 못할 수 있으므로,  
> 적용 후 실제 체감을 비교하고 필요하면 수동으로 조정하세요.

---

## 주의 사항

- 반드시 **관리자 권한**으로 실행하세요. 레지스트리 쓰기에 필요합니다.
- 한 번에 모든 장치를 변경하지 말고, GPU/오디오 등 핵심 장치부터 순차적으로 적용하세요.
- Processor Group 0의 최대 64개 논리 코어 범위만 지원합니다.
- 오버클럭·언더볼팅·백그라운드 앱 상태에 따라 결과 편차가 발생할 수 있습니다.
- **Apply, Undo Last, Factory Reset 모두 재부팅 후에 실제로 적용됩니다.**
